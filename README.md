# Health Integration Monitor

Angular 20 healthcare integration dashboard for monitoring patient records and HL7-style message flows.

## Features

- Angular 20 standalone components
- Signals and computed signals
- Patient record dashboard
- Search and reset filters
- Status filtering
- HL7 message type filtering
- Active filter summary
- Summary cards and failure rate
- Critical ORU^R01 failure alert
- Route resolver pattern
- Guards, interceptors and unit tests
